# 🌌 Aurora Glass

> A macOS-first Spicetify theme inspired by Apple's Liquid Glass.

Aurora Glass reimagines Spotify with soft frosted glass, subtle aurora gradients, and a design language that feels native on macOS.

---

## ✨ Features

- 🍎 Designed for macOS first
- 🌫️ Frosted glass interface
- 💙 Aurora blue → purple accents
- 🪟 Rounded cards and controls
- 🌙 Beautiful dark appearance

---

## 🚧 Development Status

Aurora Glass is currently in active development.

The goal is simple:

> Make Spotify feel like it was designed by Apple.
